# Selection Sort

## Time Complexity

- Best: O(n²)
- Average: O(n²)
- Worst: O(n²)

## Space Complexity

- O(1)

## Stable

No

## Notes

- Finds the minimum element and places it at the correct position.
- Performs fewer swaps compared to Bubble Sort.
- Useful when memory writes are costly.
- Not efficient for large datasets.
